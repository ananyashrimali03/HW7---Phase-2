# 113_hw7

This project is a replica of the classic fireboy and watergirl game, created using Python. The arrow keys move the fireboy character and the keys WASD move the watergirl characters. Characters can move right/left and jump onto surfaces, to collect jewels of the same type. If either one of them enters pit that are the opposite type, the game ends and 'R' can be pressed to restart the game. There are currently collision detection features and game visuals. More things to include are fixing the hazard detection, having better aesthetics, more levels, etc.
